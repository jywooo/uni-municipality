# Uni Municipality

A clean frontend-only Municipality Events Management System built with React, HTML, CSS, and JavaScript for a university thesis presentation.

## Run locally

### npm

```bash
npm install
npm run dev
```

### pnpm

```bash
pnpm install
pnpm dev
```

## Stack

- React
- Vite
- JavaScript / JSX
- Tailwind CSS
- React Router
- Mock data from `src/data/mockData.js`

## Project notes

- No backend
- No database connection
- No TypeScript in the active app
- Local mock data with browser persistence for demo use
- Presentation-ready municipal dark UI
